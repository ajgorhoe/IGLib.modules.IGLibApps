
# Project `IGLib.App`

See also [repository's README.md](../../README.md) ([on GitHub](https://github.com/ajgorhoe/IGLib.modules.IGLibApps/blob/main/README.md)).

## About `IGLib.App`

This is an extended library of application-level utilities. Utilities in this library can use reflection and do not have strict limitations on the use of external libraries.
